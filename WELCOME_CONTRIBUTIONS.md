# Welcome contributions:

Here are some currently high priority contributions that would be welcome in the library!
If you'd like to do some of these, please first comment / open an issue, so that effort is not wasted on duplicate submissions.

Issues are generally categorized as [easy good first issues](https://github.com/emilk/egui_plot/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22good%20first%20issue%22) or more [challenging issues](https://github.com/emilk/egui_plot/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22good%20challenging%20issue%22)

# Long term goal: Make all `matplotlib` examples work in `egui_plot`.

We would like to mirror all the examples from matplotlib in this library, with a similar API and rendering.
Not all functionality needed to accomplish this is yet present in `egui_plot`.

Following table summarizes either what needs to be implemented or link to an `egui_plot` example.
