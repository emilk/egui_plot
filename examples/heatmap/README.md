# Heatmap Demo

This example demonstrates how to create animated heatmaps with customizable color palettes, dimensions, and labels. It visualizes a 2D grid of values using color gradients with the Turbo colormap.

## Running

Native
```sh
cargo run -p heatmap
```

Web (WASM)
```sh
cd examples/heatmap
trunk serve
```
